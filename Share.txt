[이 파일의 용도]
윈도우 노트북과 맥북, 두 환경에서 이 과제를 이어서 작업하기 위한 진행상황 공유 파일.
Claude와의 대화 세션은 두 노트북 간에 공유되지 않으므로, 각 단계가 "확정/완료"될 때마다
이 파일을 갱신한다. 다른 노트북에서 작업을 시작할 때는 먼저 이 파일을 읽고 이어서 진행한다.
(git add / commit / push, 다른 쪽에서는 git pull 로 동기화)

===================================================
진행 방식 (합의됨)
===================================================
Goal.txt의 미션 순서대로 "단계별"로 진행. 각 단계가 끝나면 사용자 확인 후 다음 단계로 이동.
단계 순서:
  1. 7개 HTML 파일 기본 구조 (필수 시맨틱 태그)
  2. 미션1~2: CSS 분리, 폰트/색상, container, 카드/테이블 스타일
  3. 미션3~4: Flexbox/Grid 반응형 레이아웃
  4. 미션5: hover/transition/페이드인 애니메이션
  5. 미션6~8: JS (upDown, grade, bag)
  6. 미션9: 날씨 API (weatherAPI.js, realtimeInfo.js)
